# Telemetrie

<img src="https://github.com/Spaceflight-Rocketry-Giessen-e-V/Telemetrie/blob/main/pcb/Telemetrie%20v2.png" width=33%>

Das Telemetriesystem für den bidirektionalen Datenaustausch zwischen dem ASCENT II Flugcomputer und der Bodenstation ausgelegt.
Einerseits werden Sensor- und Statusdaten der Rakete in möglichst hoher Quantität übetag, während 
ist für die Übertragung von Sensor- und Statusdaten des ASCENT II Flugcomputers an die Bodenstation ausgelegt und kann zudem Telekommandos erhalten und ausführen.
